การนำเข้าตารางฐานข้อมูล Supabase

กรุณาคัดลอกโค้ด SQL ด้านล่างนี้ ไปวางในเมนู SQL Editor บนหน้าต่าง Dashboard ของ Supabase ของคุณเพื่อสร้างตารางทั้งหมด:


            
          

          

            
            
ระบบจะตรวจเช็คโครงสร้างตารางโดยอัตโนมัติ เมื่อสร้างตารางด้านบนเสร็จสิ้นและเปิด RLS แล้ว ระบบจะเปลี่ยนเป็นโหมดออนไลน์ทันทีเมื่อคุณรีเฟรชหน้าเว็บ

          

        

        

          
        

      

    


    
    

      

        

          

            
          

          
ดาวน์โหลดข้อมูลเพื่อใช้งานออฟไลน์

          
ประหยัดปริมาณการใช้เน็ตมือถือและช่วยให้ระบบทำงานเร็วขึ้น 10 เท่า

        

        
        

          
ดาวน์โหลดฐานข้อมูลที่จำเป็นเก็บไว้ในเครื่องของคุณ

          
          

            

              
              

                
โหลดข้อมูลเร็วพิเศษ (Instant Load)

                
ภาพโลโก้ พิกัดแผนที่ และลายเซ็นครูจะถูกดึงจากในเครื่องทันที ไม่ต้องรอดาวน์โหลดใหม่ทุกครั้งที่เปิดเว็บ

              

            

            
            

              
              

                
ประหยัดเน็ตมือถือ (Bandwidth Saving)

                
ลดการดึงข้อมูลรูปโปรไฟล์และลายเซ็นขนาดใหญ่ซ้ำๆ ผ่านเน็ตมือถือของคุณครู

              

            


            

              
              

                
รองรับโหมดไม่มีสัญญาณเน็ต (Offline Resilience)

                
คุณครูสามารถลงลายเซ็น ถ่ายรูป และยืนยันเวรได้ตามปกติ ระบบจะส่งขึ้นคลาวด์เมื่อเน็ตกลับมาใช้งานได้
